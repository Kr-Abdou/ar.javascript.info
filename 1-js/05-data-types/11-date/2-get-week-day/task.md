# إظهار أيام الأسبوع

قم بكتابة الدالة `getWeekDay(date)` التي تقوم بإظهار يوم الأسبوع بتنسيق قصير: 'MO'، 'TU'، 'WE'، 'TH'، 'FR'، 'SA'، 'SU'.

على سبيل المثال:

```js no-beautify
let date = new Date(2012, 0, 3);  // 3 Jan 2012
alert( getWeekDay(date) );        // يجب أن يظهر "TU"
```

# اطفال ال DOM

انظر إلى هذه الصفحة:

```html
<html>
  <body>
    <div>المستخدمون:</div>
    <ul>
      <li>جون</li>
      <li>بيت</li>
    </ul>
  </body>
</html>
```

لكل من التالي، أعطِ طريقة واحدة على الأقل لكيفية الوصول إليها:

- عقدة `<DOM <div`؟
- عقدة `<DOM <ul`؟
- العنصر `<li>` الثاني (مع Pete)؟
